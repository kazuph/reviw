# reviw リポジトリ専用メモ

## reviw ビューワ起動手順（Markdown/YAML 検証）
- 必ず同じシェルのフォアグラウンドで起動する。例: `timeout 900s node cli.cjs test-sample.md`
- AppleScript や別ターミナル経由の起動は禁止（標準出力が取得できずコメントが回収できないため）。
- 起動前に `lsof -nP -iTCP:3000 -sTCP:LISTEN` でポート衝突を確認し、必要なら既存プロセスを終了させてから実行する。
- ブラウザで「Submit & Exit」を押して終了する。終了後、同じターミナルに出る YAML の標準出力を必ず取得して報告する。
- タイムアウトは 900 秒以上を指定し、手動で中断しないこと（途中終了でコメントが破棄されるため）。
